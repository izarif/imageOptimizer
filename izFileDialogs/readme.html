izFileDialogs

This is a modified version of Native File Dialog library

Modified by Aydar Zarifullin

Building

Only Windows is supported for now

  1. Install "Visual Studio 2017" or newer
  2. Install "Desktop Development with C++" package using "Visual Studio Installer"
  3. Install "Premake"
  4. Build "LuaJIT" library
  5. Place "lua51.lib" file to "<pathToCurrentDir>\buildDeps\lua\" folder
  6. Place "lua.h", "lualib.h", "lauxlib.h", "luaconf.h" files to "<pathToCurrentDir>\buildDeps\lua\" folder
  7. Build "izNative" library
  8. Place "izn.lib" file to "<pathToCurrentDir>\buildDeps\izn\" folder
  9. Place "izn.h" file to "<pathToCurrentDir>\buildDeps\izn\" folder
  10. Open "x86 Native Tools Command Prompt for VS"
  11. Execute the following commands: 
    
> cd <pathToCurrentDir>
> premake5.exe vs2017

    You will see "izfd.vcxproj" file

  12. Open "izfd.vcxproj" file using "Visual Studio"
  13. Build as every other project
