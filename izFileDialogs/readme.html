izFileDialogs

This is a modified version of Native File Dialog library

Modified by Aydar Zarifullin

Building

Only Windows is supported for now

  1. Install Visual Studio 2017 or newer
  2. Install Desktop Development with C++ package using Visual Studio Installer
  3. Install Premake
  4. Build LuaJIT library
  5. Place lua51.lib file to <currentFolderPath>\buildDeps\lua\ folder
  6. Place lua.h, lualib.h, lauxlib.h, luaconf.h files to <currentFolderPath>\buildDeps\lua\ folder
  7. Build izNative library
  8. Place izn.lib file to <currentFolderPath>\buildDeps\izn\ folder
  9. Place izn.h file to <currentFolderPath>\buildDeps\izn\ folder
  10. Run x86 Native Tools Command Prompt for VS
  11. Execute the following commands: 
    
> cd <currentFolderPath>
> <premakeBinPath> vs2017

    Where <currentFolderPath> is something like C:\Users\zar-s\projects\imageOptimizer\izFileDialogs

    Where <premakeBinPath> is something like C:\Users\zar-s\soft\premake5.exe

    After executing these commands you will see izfd.vcxproj file

  12. Open izfd.vcxproj file using Visual Studio
  13. Build as any other project
