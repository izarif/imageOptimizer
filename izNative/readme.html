izNative

Library that contains platform-specific code

Building

Only Windows is supported for now

  1. Install "Visual Studio 2017" or newer
  2. Install "Desktop Development with C++" package using "Visual Studio Installer"
  3. Install "Premake"
  4. Build "LuaJIT" library
  5. Place "lua51.lib" file to "<pathToCurrentDir>\buildDeps\lua\" folder
  6. Place "lua.h", "lualib.h", "lauxlib.h", "luaconf.h" files to "<pathToCurrentDir>\buildDeps\lua\" folder
  7. Open "x86 Native Tools Command Prompt for VS"
  8. Execute the following commands: 
    
> cd <pathToCurrentDir>
> premake5.exe vs2017

    You will see "izn.vcxproj" file

  9. Open "izn.vcxproj" file using "Visual Studio"
  10. Build as every other project
