imageOptimizer

Программа для сжатия изображений формата png, jpg и gif

Сжатие происходит без потерь, то есть качество изображений не ухудшается

Системные требования

  • Операционная система: Windows Vista или новее

Использование

Примечание: Перед использованием убедитесь что у вас достаточно свободного места для хранения сжатых копий изображений

Примечание: Сжатое изображение может весить больше оригинала (если это произошло значит данное изображение невозможно сжать)

Распакуйте архив imageOptimizer.zip в отдельную папку

Откройте папку с программой и запустите файл imageOptimizer.exe

Из выпадающего списка выберите предпочитаемый язык и нажмите на кнопку Ок

Для того чтобы сжать изображение нажмите на кнопку Добавить

Выберите один или несколько файлов подходящего формата и нажмите на кнопку Открыть

В главном окне программы, в списке, отобразятся пути к выбранным файлам

Чтобы убрать файл из списка кликните по нему и нажмите на кнопку Убрать (можно выбрать несколько файлов если удерживать клавишу ctrl при клике)

Чтобы начать процесс сжатия, нажмите на кнопку Начать

Ждите, пока не появится окно с текстом Процесс завершен

После завершения процесса сжатые версии изображений будут сохранены в папке программы, в подпапке optimized

После завершения процесса в папке программы появится файл report.txt с информацией о том, насколько хорошо сжались изображения

Если вы хотите прервать процесс, нажмите на кнопку Отменить

Если вы по каким-либо причинам завершили работу программы через Диспетчер задач, тогда необходимо вручную завершить следующие процессы:

  • optipng
  • jpegtran
  • gifsicle

Не редактируйте файл cfg.json без необходимости
