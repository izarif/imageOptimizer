imageOptimizer

Program for compressing PNG, JPG, and GIF images

The compression is lossless, meaning the image quality is not compromised

System requirements

  • Operating system: Windows Vista or newer

Usage

Note: Before using, make sure that you have enough free space to store compressed copies of images

Note: The compressed image can be larger than the original (if this happens, it means that this image cannot be compressed)

Unpack the "ImageOptimizer.zip" archive to a separate folder

Open the program folder and run the "imageOptimizer.exe" file

Select your preferred language from the drop-down list and click the "Ok" button

To compress the image, click "Add" button

Select one or more files of a suitable format and click the "Open" button

The paths to the selected files will be displayed in the main window of the program, in thw list

To remove a file from the list, click on it and click the "Remove" button (you can select several files if you hold down the "ctrl" key when clicking)

To start the compression process, click the "Start" button

Wait until a window appears with the text "Process completed"

Once the process is complete, the compressed versions of the images will be saved in the program folder, in the "optimized" subfolder

Once the process is complete, a file named "report.txt" will be created in the program folder, containing information on how well the images were compressed

If you want to cancel the process, click the "Cancel" button

If, for any reason, you have terminated the program using "Task Manager" you must manually terminate the following processes:

  • optipng
  • jpegtran
  • gifsicle

Do not edit the "cfg.json" file unless necessary
